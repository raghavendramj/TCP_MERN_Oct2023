Bootstrap 5 Text/Typography

Bootstrap 5 uses a default font-size of 1rem (16px by default), and its line-height is 1.5.

Elements have margin-top: 0 and margin-bottom: 1rem (16px by default).

Highlight Text

Use the mark element (or the .mark class) to highlight text.

Abbreviations

The abbr element is used to mark up an abbreviation or acronym:

The WHO was founded in 1948.

Description Lists

The dl element indicates a description list:

Coffee
- black hot drink
Milk
- white cold drink

Code Snippets

Inline snippets of code should be embedded in the code element:

The following HTML elements: public static void main(String args[]), section, and div defines a section in a document.
